# Agent-Based Model: Social Network Formation

NetLogo agent-based model simulating how social networks form based on warmth and competence cues, grounded in the Stereotype Content Model and Coleman's Boat framework. Built as part of my MSc in Social Data Science at UCD.

## What this shows
- Agents form social ties based on perceived warmth and competence
- Simulation run across 1,650 runs and 63 parametric configurations to test how different parameter combinations affect network structure
- Explores how micro-level perception biases (warmth/competence stereotypes) translate into macro-level network patterns

## Why it matters
This connects social psychology theory (stereotype formation) to computational modelling, showing how individual-level biases can shape collective social structures over time.
